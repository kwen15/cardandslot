About the Project

This app studies the Filipino Perya Color Game using Monte Carlo simulation. It contrasts a fair version with multiple tweaks (altered probabilities or payouts) to illustrate how a small change can create a house edge.

  • Fair game baseline with equal probabilities.
  • Tweaked and weighted versions that shift expected value.
  • Modified payout and normal distribution multipliers.
  • Lucky 9 card-based variant.
What the simulation shows

For each configuration we track win rate, player ROI, house ROI, and cumulative profit over time. Visual charts make it easy to see divergence between fair and tweaked games.
